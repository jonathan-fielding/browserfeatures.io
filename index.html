BrowserFeatures.io

Community project to provide an API to enable you to include browser feature information in your webpages.

API

/v1/feature/:feature-name

Retrieve data for a given feature

Example Request: http://127.0.0.1:9000/v1/feature/border-radius

If no API is found suggestions will be offered

Example of a failed request: http://127.0.0.1:9000/v1/feature/border

{
    "error": "api not found",
    "suggestions": ["border-image", "border-radius"]
}
